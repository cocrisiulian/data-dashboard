
# Data Dashboard

This project is a modern dashboard web application for uploading CSV files, creating interactive charts, and building custom dashboards.

## Features

- User authentication
- File upload and storage
- Data visualization (charts, dashboards)
- Plan management

## Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install` or `npm install`
3. Set up your `.env.local` with your Supabase credentials
4. Run the development server: `pnpm dev` or `npm run dev`

## Project Structure

- `src/app/` – Main Next.js app pages
- `src/lib/` – Utilities, Supabase client, actions
- `src/components/` – UI components
