# Lab 2: Architecture Overview

- Client-server architecture: Next.js frontend, Supabase backend (auth, DB, storage).
- Communication via RESTful APIs and Supabase client libraries.
- PostgreSQL database with modular schema and RLS.

## Component Diagram (Textual)
- Client (Next.js): UI, user auth, file upload, dashboard/chart management.
- Supabase: Auth, DB, Storage, RLS.
